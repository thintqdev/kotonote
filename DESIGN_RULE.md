# UI Style Rules

Overall aesthetic:

- Handmade sketchbook
- Japanese study notebook
- Pencil sketch + paper texture
- Cozy vintage learning vibe

Colors:

- Background: #f7f1e8
- Ink: #2b2b2b
- Accent green: #a8b58a
- Accent beige: #d8c3a5

Components:

- Borders slightly imperfect
- Use dashed/divider hand-drawn style
- Rounded corners soft
- Add tape/sticky-note decorations
- Avoid modern SaaS look

Typography:

- Mix handwritten font + readable sans-serif
- Japanese furigana support

Effects:

- Paper texture overlay
- Coffee stain PNG
- Pencil shadow
- Light noise
